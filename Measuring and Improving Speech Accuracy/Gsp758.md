## 🌐 **Quick Start Guide:**

**Launch Cloud Shell:**  
 Start your Google CloudShell session by [clicking here](https://console.cloud.google.com/home/dashboard?project=&pli=1&cloudshell=true).

```bash
curl -LO https://raw.githubusercontent.com/Cloud-Hustlers/Lab_Solutions/refs/heads/main/Measuring%20and%20Improving%20Speech%20Accuracy/gsp758.sh

sudo chmod +x gsp758.sh

./gsp758.sh

```
